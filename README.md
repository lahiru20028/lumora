# Lumora Candles E-Commerce Platform

A high-performance, full-stack e-commerce web application designed for a premium candle brand. Built using the **MERN stack** (MongoDB, Express, React, Node.js) and following a scalable **2-tier architecture**.

## üåü Features

### User Roles & Authentication
* **Secure Auth:** JWT-based signup and login flow for customers.
* **Customer Profile:** Manage account details and track personal order history.

### Shopping Experience
* **Dynamic Catalog:** Browse candle collections by categories like Glass, Flower, and Seasonal with real-time updates.
* **Product Customization:** Interactive interface to select fragrances (e.g., Lavender, Cinnamon) and adjust quantities.
* **Review System:** Full CRUD functionality for product reviews, allowing users to upload images and share feedback.

### Payments & Checkout
* **Stripe Integration:** Secure financial transactions using the **Stripe API** for credit and debit cards.
* **Form Validation:** Accurate shipping detail collection with real-time input validation.

### Administration
* **Admin Dashboard:** Powerful management suite to track total stock, monitor sales, and manage order lifecycles.
* **Inventory Management:** Easily add, edit, or delete products and manage stock levels directly from the UI.
